Адаптивная кроссбраузерная семантическая верстка многостраничного сайта 1eska.ru
«Простые решения» — компания, которая занимается автоматизацией работы с программами 1С. Это партнёр фирмы 1С, цель — сделать ведение учёта и бизнеса проще, а работу с 1С — понятной и удобной.

Стек HTML5 CSS3 JavaScript PHP
Методология BEM • Flex/Grid
Адаптация Desktop First • Mobile First
Верстка
Сделана кроссбраузерная адаптивная семантическая верстка по макету в Figma
Адаптивность
Сделан адаптив под мобильные и планшетные устройства. Гибкая верстка, перестраивается в зависимости от разрешения экрана.
Интеграция
Интеграция верстки под CMS 1С Битрикс: вынесен header и footer отдельно, выделена контентная часть, созданы компоненты для различных элементов и блоков страниц
Сборка
Созданы все необходимые инфоблоки и контентные блоки в админке Битрикс, выполнена сборка, т.е. форматирование и вывод данных из админки в шаблоны

Особенности реализации

Технические детали, которые демонстрируют уровень владения вёрсткой

Интерактивные состояния

:hover, :focus, :active, transition, кастомные анимации, микроинтеракции. Все состояния проработаны.

Адаптивная сетка

CSS Grid + Flexbox, 5 брейкпоинтов, mobile-first подход. Проверено на 15+ устройствах.

Доступность (a11y)

Семантическая верстка, ARIA-метки, управление с клавиатуры, контрастность WCAG AA.

Оптимизация

Critical CSS, lazy loading, оптимизированные шрифты, минимальный размер бандла.

BEM-методология

Читаемая, масштабируемая структура классов. Независимые компоненты, переиспользование.

Браузерная совместимость

Chrome, Firefox, Safari, Edge, Opera. Автопрефиксы, fallback-стили.
